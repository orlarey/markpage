% Theorem-like environments matching the ::: admonitions (§17 of

% proposition / corollary numbered in the same sequence as the

A \texttt{"quoted caption"} after the language tag wraps any block in an
auto-numbered \texttt{<figure>}. Each kind keeps its own counter:
\texttt{Figure N} for visual blocks, \texttt{Tableau N} for data, \texttt{Listing N} for
code, \texttt{Algorithme N} for pseudocode.


A captioned table:


\begin{center}
\begin{tabular}{lll}
  \toprule
  Équipe & Effectif & Budget \\
  \midrule
  Backend & 12 & 1.2M \\
  Frontend & 8 & 0.7M \\
  Data & 5 & 0.5M \\
  \bottomrule
\end{tabular}
\end{center}


A captioned tree diagram:


\begin{lstlisting}
S
  NP
    Det
    N
  VP
    V
    NP
      Det
      N
\end{lstlisting}


A captioned code listing:


\begin{lstlisting}[language=Python]
def median(xs):
    s = sorted(xs)
    n = len(s)
    return s[n // 2] if n % 2 else (s[n // 2 - 1] + s[n // 2]) / 2
\end{lstlisting}


A captioned algorithm:


\begin{lstlisting}
Input: array A, value x
for i from 0 to length(A) - 1 do
  if A[i] = x then
    return i
  end
end
return -1
\end{lstlisting}
