A ::: style block applies a fixed allowlist of typographic overrides (colour, size, font, weight, alignment, italic, underline, line-height) to its Markdown body — never arbitrary CSS or HTML.

A blue, 22-point, centred, bold title

The body is ordinary Markdown, and the style cascades into it:

This whole block is Lora grey.

::: style size=20pt italic A nested aside — still Lora grey, but larger and italic (the inner block wins).

:::
