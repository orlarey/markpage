\begin{lstlisting}
Input: array A of length n
Output: A sorted in place
for i from 1 to n - 1 do
  for j from 0 to n - i - 1 do
    if A[j] > A[j + 1] then
      swap A[j] and A[j + 1]
    end
  end
end
return A
\end{lstlisting}


A second algorithm in the same document keeps incrementing the
counter, so the next caption reads \texttt{Algorithme 2}:


\begin{lstlisting}
Input: sorted array A, target x
Output: index of x in A, or -1
lo := 0
hi := length(A) - 1
while lo <= hi do
  mid := (lo + hi) / 2
  if A[mid] = x then
    return mid
  elif A[mid] < x then
    lo := mid + 1
  else
    hi := mid - 1
  end
end
return -1
\end{lstlisting}
