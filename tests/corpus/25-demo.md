# Demo — source beside its rendering

A `demo` fence shows a Markdown snippet and its live rendering side by
side — handy for tutorials and feature showcases. The body is rendered
through the same pipeline as the document.

```demo
A paragraph with **bold**, *italic* and `inline code`, plus inline
math $a^2 + b^2 = c^2$.
```

A demo of a callout:

```demo
::: tip
Pick the simpler block when two fit.
:::
```

A demo of a list:

```demo
1. First
2. Second
   - nested
   - items
```
